## 📌 Project Overview
This project is a Machine Learning-based House Price Prediction system built using the Boston Housing dataset. The model analyzes multiple features such as crime rate, number of rooms, tax rate, and other socio-economic factors to predict the price of a house.

It demonstrates a complete end-to-end regression workflow in Python, from data preprocessing to model evaluation.

---

## 🎯 Objective
- Build a regression model to predict house prices
- Understand the application of boosting algorithms
- Gain hands-on experience with real-world datasets

---

## 🛠️ Technologies & Libraries Used
- Python
- NumPy
- Pandas
- Scikit-learn
- XGBoost
- Matplotlib (optional)

---

## ⚙️ Machine Learning Workflow
1. **Data Collection**
   - Used the Boston Housing dataset

2. **Data Preprocessing**
   - Checked for missing values
   - Feature selection and understanding correlations

3. **Train-Test Split**
   - Used `train_test_split` to divide the dataset into training and testing sets

4. **Model Building**
   - Implemented **XGBoost Regressor** for prediction

5. **Model Evaluation**
   - Evaluated the model using performance metrics like R² score and error metrics

## 📊 Features
- Predicts house prices based on input features
- Uses advanced boosting algorithm (XGBoost)
